¿Qué es VueJS?

Es un framework progresivo de javascript para construir interfaces web interactivas.

Creador de Vue

Evan You

Historia

Evan You que trabajaba en Google realizando prototipos y en el core del Framework de Meteor, hasta que pensó en otra forma de hacer una opción más fácil que abarcara las necesidades a la hora de hacer prototipos. Así surgió Vue en el 2014, desde entonces ha tenido una gran evolución y sigue creciendo en su versión 2.

Características

  • Fácil de Aprender
  • Sistema de Data Binding
  • Sistema de Componentes
  • Reactivo
  • Optimizado: su core ocupa 74KB(Bastante Liviano)
  • Excelente Librería para hacer peticiones http, validación de formulario, etc.
  • Escalable
  • Licencia MIT

Patrón MVVM

Consiste en pasar datos desde el código javascript "MODELO" al código html "VISTA", de una manera sencilla, sin tener que tratar directamente con las etiquetas html desde javascript.

Instancia de Vue

new Vue();

Propiedades de Vue

var app = new Vue({
    el: '#app',

    data: {

        ciudad: 'Barcelona',
        pais: 'Venezuela',

    },

    computed: {
        dondeEstoy: function () {
            return 'Estoy en ' + this.ciudad + ' ' + this.pais;
        }
    },

    methods: {
        mensajeCiudad: function () {
            console.log('Soy la Ciudad de ' + this.ciudad);
        }
    },

    mounted: function () {
        this.mensajeCiudad();
    }
});

Propiedad el

  • Esta Propiedad determina el TAG HTML que operará nuestra instancia Vue.
  • Cada Instancia de Vue apunta a un solo nodo html y sus hijos.
  • Podemos utilizar como selector una clase/id/etiqueta.
var app = new Vue({
    //Propiedad el
    el: '#app' //Selector por ID
});

Propiedad data

Es un objeto con propiedades que podemos utilizar en nuestra instancia.

var app = new Vue({
    el: '#app',
    //Propiedad Data
    data: {
        ciudad: 'Barcelona',
        pais: 'Venezuela',
    },
});

Hola Mundo

Directivas

V-TEXT

Muestra contenido textual dentro de un elemento.

<div id="app">
    <h1 v-text="mensaje"></h1>
</div>

V-HTML

Muestra contenido textual dentro de un elemento(Acepta Código HTML).

var app = new Vue({
    el: '#app',
    data: {
        titulo: '<p>Curso de VueJS</p>'
    }
});
<div id="app">
    <h1 v-html="titulo"></h1>
</div>

V-SHOW

Permite Mostrar u Ocultar contenido de DOM.

var app = new Vue({
    el: '#app',
    data: {
        mostrarTitulo: true,
        mostrarSubtitulo: false
    }
});
<div id="app">
    <h1 v-show="mostrarTitulo">Curso de VueJS</h1>
    <h3 v-show="mostrarSubtitulo">Framework de JavaScript</h3>
</div>

V-IF

Bloque Condicional

<div id="app">
    <h1 v-if="mostrarTitulo">Curso de VueJS</h1>
    <h3 v-if="mostrarSubtitulo">Framework de JavaScript</h3>
    <h3 v-else>No Tengo SubTítulo</h3>
</div>

V-BIND

Permite vincular un Atributo HTML a un valor que tenemos en nuestro Modelo.

var app = new Vue({
    el: '#app',
    data: {
        btnDisabled: true
    }
});
<div id="app">
    <!-- v-bind:nombreAtributo -->
    <button v-bind:disabled="btnDisabled">Enviar Datos</button>
    <!-- El Atajo : -->
    <button :disabled="btnDisabled">Cancelar Datos</button>
</div>

V-ON

Manejar Evento

var app = new Vue({
    el: '#app',
    data: {
        msj: 'Buenos Días'
    },
    methods: {
        mensajeAlert: function(mensaje, event) {
            console.log(event);
            alert(mensaje);
        }
    }
});

V-ON

Manejar Evento

<div id="app">
    <button @click="mensajeAlert('Hola Programate', $event)">Programate @</button>
</div>
