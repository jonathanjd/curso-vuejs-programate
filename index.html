¿Qué es VueJS?

Es un framework progresivo de javascript para construir interfaces web interactivas.

Creador de Vue

Evan You

Historia

Evan You que trabajaba en Google realizando prototipos y en el core del Framework de Meteor, hasta que pensó en otra forma de hacer una opción más fácil que abarcara las necesidades a la hora de hacer prototipos. Así surgió Vue en el 2014, desde entonces ha tenido una gran evolución y sigue creciendo en su versión 2.

Características

  • Fácil de Aprender
  • Sistema de Data Binding
  • Sistema de Componentes
  • Reactivo
  • Optimizado: su core ocupa 74KB(Bastante Liviano)
  • Excelente Librería para hacer peticiones http, validación de formulario, etc.
  • Escalable
  • Licencia MIT

Patrón MVVM

Consiste en pasar datos desde el código javascript "MODELO" al código html "VISTA", de una manera sencilla, sin tener que tratar directamente con las etiquetas html desde javascript.

Instancia de Vue

new Vue();

Propiedades de Vue

var app = new Vue({
    el: '#app',

    data: {

        ciudad: 'Barcelona',
        pais: 'Venezuela',

    },

    computed: {
        dondeEstoy: function () {
            return 'Estoy en ' + this.ciudad + ' ' + this.pais;
        }
    },

    methods: {
        mensajeCiudad: function () {
            console.log('Soy la Ciudad de ' + this.ciudad);
        }
    },

    mounted: function () {
        this.mensajeCiudad();
    }
});

Propiedad el

  • Esta Propiedad determina el TAG HTML que operará nuestra instancia Vue.
  • Cada Instancia de Vue apunta a un solo nodo html y sus hijos.
  • Podemos utilizar como selector una clase/id/etiqueta.
var app = new Vue({
    //Propiedad el
    el: '#app' //Selector por ID
});

Propiedad data

Es un objeto con propiedades que podemos utilizar en nuestra instancia.

var app = new Vue({
    el: '#app',
    //Propiedad Data
    data: {
        ciudad: 'Barcelona',
        pais: 'Venezuela',
    },
});

Directivas

V-TEXT

Muestra contenido textual dentro de un elemento.

<div id="app">
    <h1 v-text="mensaje"></h1>
</div>

V-HTML

Muestra contenido textual dentro de un elemento(Acepta Código HTML).

var app = new Vue({
    el: '#app',
    data: {
        titulo: '<p>Curso de VueJS</p>'
    }
});
<div id="app">
    <h1 v-html="titulo"></h1>
</div>

V-SHOW

Permite Mostrar u Ocultar contenido de DOM.

var app = new Vue({
    el: '#app',
    data: {
        mostrarTitulo: true,
        mostrarSubtitulo: false
    }
});
<div id="app">
    <h1 v-show="mostrarTitulo">Curso de VueJS</h1>
    <h3 v-show="mostrarSubtitulo">Framework de JavaScript</h3>
</div>

V-IF

Bloque Condicional

<div id="app">
    <h1 v-if="mostrarTitulo">Curso de VueJS</h1>
    <h3 v-if="mostrarSubtitulo">Framework de JavaScript</h3>
    <h3 v-else>No Tengo SubTítulo</h3>
</div>

Bus $on

<template>
<div class="box effect">
    <h1>{{ title }} - {{ myApellido }}</h1>
    <h3>Mi edad: {{ edad }}</h3>
    <button @click="cambiarTitlePadre">Cambiar Title Componente Padre</button>
</div>
</template>
<script>
import { eventBus } from '../main.js';
export default {
data: function() {
    return {
    title: 'Componente Hijo',
    mutationMyEdad: this.myEdad,
    edad: 20
    };
},
created() {
    eventBus.$on('cambiarEdad', edad => {
    this.edad = edad;
    });
},
props: {
    myApellido: {
    type: String,
    required: false,
    default: 'Apellido Papá'
    },
    myEdad: {
    type: Number
    }
},
methods: {
    cambiarTitlePadre: function() {
    this.mutationMyEdad += 50;
    this.$emit('cambiarEdadPapa', this.mutationMyEdad);
    }
}
};
</script>

Qué es?

Axios

Qué es?

Es un cliente HTTP basado en Promesas para Javascript, el cual puede ser utilizado tanto en tu aplicación Front-end, como en el Back-end con Nodejs. Utilizando Axios, es muy sencillo enviar peticiones a endpoints REST y realizar operaciones CRUD.

Feature

  • Realizar peticiones XMLHttpRequest (Ajax) desde el navegador de una manera sencilla.
  • Soporta el API de Promesas.
  • Intercepta peticiones y respuestas.
  • Transforma la información de las peticiones y respuestas.
  • Cancela peticiones.
  • Transforma automáticamente la información en formato JSON.
  • Soporta protección del lado del cliente contra ataques CSRF (Cross-site request forgery).

Instalación

npm install axios

Enviando un Post Request

<script>
import axios from 'axios';
export default {
data() {
    return {
    snackbar: false,
    color: 'success',
    mode: '',
    timeout: 6000,
    textSucces: 'Mensaje Registrado',
    inputMensaje: '',
    mensajes: ['Mensaje 123456', 'Mensaje 987654', 'Mensaje 987455']
    };
},
computed: {
    getMensajes() {
    return this.mensajes;
    },
    validarButton() {
    return this.inputMensaje == '';
    }
},
methods: {
    guardar() {
    let formData = {
        contenido: this.inputMensaje
    };
    axios
        .post('https://my-vue-axios.firebaseio.com/mensaje.json', formData)
        .then(response => {
        console.log(response);
        this.snackbar = true;
        })
        .catch(error => {
        console.log(error);
        this.snackbar = true;
        this.color = 'danger';
        this.textSucces = 'Se Produjo un Problema en el Proceso';
        });
    this.limpiarInput();
    },
    limpiarInput() {
    this.inputMensaje = '';
    }
}
};
</script>
